# 💰 Expense Tracker (Full Stack)

A full-stack Expense Tracker application built using Spring Boot and React (Vite). It allows users to manage expenses with filtering, editing, and real-time calculations.

---

## 🚀 Features

- Add, update, delete expenses
- Filter by category and date range
- Total expense calculation
- Clean and modern UI using Tailwind CSS
- REST API integration

---
## 🔥 Highlights

- Implemented full CRUD operations
- Used DTO and validation for backend
- Designed responsive UI with Tailwind CSS
- Applied filtering logic and dynamic calculations
---

## 🛠️ Tech Stack

### Backend:
- Java
- Spring Boot
- Spring Data JPA
- MySQL

### Frontend:
- React (Vite)
- Tailwind CSS
- Axios

---

## 📂 Project Structure
backend/ → Spring Boot APIs
frontend/ → React UI


---

## ⚙️ How to Run

### Backend:
1. Open backend folder
2. Run Spring Boot application
3. Ensure MySQL is running

### Frontend:
1. Go to frontend folder
2. Run:

npm install
npm run dev
